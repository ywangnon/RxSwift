## 장점

1. 자유로운 구성 가능
2. 재사용 가능: 자유로운 구성이 가능하기 때
3. 선언적: 데이터만 변경되기 때문
4. 간결함:  추상화 수준 향상
5. 안정적: Rx 코드는 유닛 테스트 되었기 때문
6. 상태 저장 감소: 데이터의 흐름으로 모델링 되었기 때문
7. 누수 없음: 리소스 관리 용이

> RxSwift를 사용하는 것은 단순하고 직관적인 작성이 가능하다.
