## Rx를 사용하는 이유

Rx를 사용했을 때, 좋은 이유를 찾아보면 여러가지 이유가 나옵니다. 그 중에서 내가 가장 와 닿은 이유를 적어보겠습니다.

1. **코드의 간결화**
   * 비동기나 복잡한 코드를 Rx를 사용하여 간결하게 작성할 수 있음
2. **비동기 코드의 명확**
   * DispatchQueue나 OperationQueue를 사용하여 데이터가 어떻게 흘러갈지 복잡하게 제어하지 않아도 됨
3. **콜백처리**
   * 콜백처리를 자연스럽게 할 수 있게&#x20;
